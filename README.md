# NoonVision â€“ AI-Powered Object Detection for the Visually Impaired

NoonVision is a web-based AI application designed to assist visually impaired individuals.  
It detects objects in real time using your webcam, shows bounding boxes, lists detected objects, and **speaks them aloud** using AI-generated voice.  

**Tech Stack:**  
- Gradio  
- YOLOv8 (Ultralytics)  
- gTTS for voice  
- Python 3  
- Hugging Face Spaces

**How to Use:**  
1. Open the app in your browser.  
2. Allow camera access.  
3. Point the webcam at an object.  
4. The app shows bounding boxes, lists detected objects, and plays audio describing the objects.
