# NoonVision â€“ AI-Powered Object Detection for the Visually Impaired

NoonVision is a web-based AI application designed to assist visually impaired individuals.  
It detects objects in real time using your webcam and speaks/labels them to help users understand their surroundings.  

**Tech Stack:**  
- Gradio  
- YOLOv8 (Ultralytics)  
- Python 3  
- Hugging Face Spaces

**How to Use:**  
- Open the app in your browser  
- Allow camera access  
- Point the webcam at an object  
- The app shows bounding boxes and lists detected objects
